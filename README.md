# God.Client

El proyecto es una aplicacion de Reactjs, solo hay que hacer
```
  npm install
  npm start
```
y debe funcionar,
La aplicacion asume que el backend esta en el **https://localhost:5001** que es  el que tiene configurado por defecto el backend de Net Core.
